Calendar Solved Problems UI Test

This test verifies that solved problems appear on the correct dates in the calendar interface.

Test 1: Fetch Solved Problems Data

This will fetch solved problems and show their solve dates.

Test 2: Calendar Date Distribution

This will show how solved problems are distributed across calendar dates.

Test 3: Day Detail View Simulation

This will simulate clicking on a calendar cell and show what problems appear.

Test 4: Calendar Service Integration

This will test the calendar service endpoints that the frontend uses.
